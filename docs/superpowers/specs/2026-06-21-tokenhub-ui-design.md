# LLM Gateway — TokenHub 风格 UI 重设计

**日期**: 2026-06-21  
**范围**: 三个用户端页面重设计：模型广场、用量监控、API Key 管理  
**参考**: 腾讯云 TokenHub 控制台  
**技术栈**: Vue 3 + Naive UI + Vite（现有项目，不引入新依赖）

---

## 设计方向

TokenHub 结构 + LLM Gateway 自身配色（主色 `#006EFF`）。借鉴 TokenHub 的信息架构和组件布局，不做生硬外观复制，保留项目品牌感。

---

## 公共 Shell

所有三个页面共享同一套页面框架：

- **左侧导航栏**（宽 200px，白底）：Logo 区 + 导航菜单项，当前页高亮（左侧蓝色竖条 + 浅蓝背景）
- **顶部栏**（高 52px，白底，底部细线分隔）：页面标题 + 右侧功能区（余额 chip / 操作按钮）
- **主内容区**：内边距 `24px 28px`，浅灰背景 `#f5f6fa`

导航菜单顺序：仪表盘 / 模型广场 / 用量监控 / API Key / Playground / 充值 / 设置

## 页面一：模型广场（`/models`）

### 布局结构

```
顶部过滤栏（供应商标签 + 搜索框）
│
├── 精选推荐区（2 列大卡片）
│   每张卡：渐变色 Banner（含模型 logo + 名称 + 副标题）+ 内容区（标签 + 描述 + 价格元数据）
│
└── 全部模型区（3 列小卡片网格）
    每张卡：左侧方形渐变 icon + 名称/供应商 + 描述（2行截断）+ 底部价格标签
```

### 交互

- **过滤标签**：按供应商单选过滤（全部 / Anthropic / OpenAI / DeepSeek / Google / 更多），支持搜索框模糊匹配
- **卡片点击**：路由跳转 `/models/:id` 详情页（详情页不在本次范围内，跳转目标为占位路由）
- **悬停效果**：卡片 `box-shadow` + 边框色变蓝

### 数据来源

复用现有 `gatewayApi.js` 的模型列表接口，字段映射：`name`、`description`、`provider`、`tags`、`input_price`、`output_price`

### 精选推荐

默认精选 `claude-sonnet-4-5` 和 `deepseek-r1` 两个模型（在 `Models.vue` 顶部以常量定义 `FEATURED_MODEL_IDS`）。其余模型按接口返回顺序展示。如接口中不存在该模型 ID，精选区自动降级为空，不报错。

## 页面二：用量监控（`/usage`，重设计现有 Usage.vue）

### 布局结构

```
时间选择栏（今日 / 昨日 / 近7天 / 近30天 / 自定义日期范围）
│
├── 4 格统计卡（消费 / 请求数 / Tokens / 错误率，含环比涨跌）
│
├── 趋势图表区（2 列）
│   ├── 左：每日消费折线图（单线，带渐变填充）
│   └── 右：按模型分组请求量折线图（多线，每模型一色）
│
└── 模型消耗明细表
    列：模型名 / 请求数 / 消耗 Tokens / 占比进度条 / 费用
    排序：按费用降序
```

### 时间范围

默认展示近 7 天。自定义日期范围通过两个日期 Picker 输入，最大跨度 90 天。

### 统计卡环比

与上一个同等时长区间对比，涨幅绿色（↑），降幅红色（↓），误差率例外（降为绿色）。

### 图表实现

复用现有 Dashboard.vue 的 SVG 折线图逻辑，不引入 echarts 等新依赖。多线图通过不同 stroke 颜色区分，最多同时展示 5 条线（按请求量取 Top 5 模型）。

### 数据来源

复用 `api.js` 的 `/reports/usage` 和 `/reports/trend` 接口，参数增加 `start_date` / `end_date` 用于自定义范围。

### 管理端版本（`/admin/user-usage` 增强）

同样布局，数据切换为全平台。明细表额外增列：用户数 / P95 延迟 / 错误次数。本次实现仅覆盖用户端，管理端作为后续迭代。

## 页面三：API Key 管理（`/keys`，重设计现有 Keys.vue）

### 布局结构

```
顶部汇总卡（3 格：Key 总数 / 有效 Key 数 / 本月总调用量）
│
├── 工具栏（搜索框 + 状态筛选 + 分组筛选 + 批量操作按钮）
│
└── 表格
    列：复选框 / Key名称+前缀掩码 / 状态徽章 / 配额进度条 / 最后调用 / 到期时间 / 操作
    分页：每页 20 条
```

### 状态徽章

| 状态 | 颜色 |
|------|------|
| 有效 | 绿色 `#059669` |
| 已过期 | 橙色 `#b45309` |
| 已撤销 | 红色 `#dc2626` |

### 配额进度条

- 使用率 < 80%：蓝色 `#006EFF`
- 使用率 ≥ 80%：橙色 `#f59e0b`，文字加粗
- 无配额限制时显示"不限"

### 操作按钮

- 有效 Key：复制 / 编辑 / 撤销
- 已撤销 Key：查看（整行置灰 60% 透明度）
- 已过期 Key：复制 / 删除

### 新建 Key

点击右上角"+ 新建 API Key"按钮，弹出 Modal（保持现有 Keys.vue 的表单字段和校验逻辑，仅更新视觉样式）。

### 批量操作

勾选多行后，批量操作按钮激活，支持：批量撤销 / 批量删除。
